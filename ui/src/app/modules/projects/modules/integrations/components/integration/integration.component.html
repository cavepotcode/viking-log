Step 1 : Create a Project Logs project

Before you can add Project Logs to your app, you need to create a new Project Logs project to connect to your app. Please in the left panel click on New Project.

Step 2 : Copy AppKey

To connect your app with Project Logs , you need to copy the App key. Please in the left panel click more_vert and then copy App-key vpn_key
