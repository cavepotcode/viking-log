Step 1 : Create a Project Logs project

Before you can add Project Logs to your app, you need to create a new Project Logs project to connect to your app. On the left menu you will see a link to create your projects.

Step 2 : Copy AppKey

After you create the project you will need to get the api key in order to develop your integrations. On the left menu click more_vert and then App-key vpn_key
